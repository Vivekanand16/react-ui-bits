import { Playground, Props } from 'docz'

# RadioButton

<Props of={RadioButton} />

## Basic usage

<Playground>
    <RadioButton name="gender" value="male" label="Male" />
</Playground>

## Selected RadioButton

To select a radiobutton, pass value of the radiobutton to `checked` prop.

<Playground>
    <RadioButton name="gender" value="male" label="Male" checked="male" onChange={() => {}} />
    <RadioButton name="gender" value="female" label="Female" checked="male" onChange={() => {}} />
</Playground>

## Height & Width

`Height` and `Width` props are used to change radiobutton size.

<Playground>
    <RadioButton name="gender" value="male" label="Male" height="30px" width="30px" />
</Playground>
