# Header

## Basic usage

The justify property is used to align header items and it accepts all values of justify-content property of flex-box. The default value is *space-between*.

        logo="//avatarfiles.alphacoders.com/205/205044.jpg" 
        logoalt="alt text" 
        logotext="Logo"
        justify="center"
    />
</Playground>
